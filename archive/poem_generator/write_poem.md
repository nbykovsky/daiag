# Write Poem

Inputs:
- `n`: ${N}
- `poem_path`: ${POEM_PATH}

Instructions:
1. Write a poem with exactly ${N} lines.
2. Each line must be a complete poetic phrase or sentence.
3. Write the poem to `${POEM_PATH}`, replacing any existing content.

Outputs:
- Write/update: ${POEM_PATH}
- Return JSON with keys:
  - `poem_path`: set to `${POEM_PATH}`

Do not wrap the JSON in Markdown fences.
