# Write a 10-Row Poem

Write a creative poem with exactly 10 rows and save it to the specified file.

Requirements:

1. Create a poem with exactly 10 rows (lines).
2. The poem should be original, coherent, and creative.
3. Each line should be a complete thought or part of a natural progression.
4. Save the entire poem to `${OUTPUT_PATH}`.

Outputs:

- `${OUTPUT_PATH}` — the file containing the complete 10-row poem

Return the following JSON with no other text:

```json
{
  "poem_path": "${OUTPUT_PATH}"
}
```

Do not wrap the JSON in Markdown fences.
